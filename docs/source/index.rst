Welcome to Ply's documentation!
===============================
** Ply ** is an adaptive, "community-grouped" software suite to create highly customisable online communities. Profiles are created by "users" to join "communities" that can represent specific groups or communities of people. 
Ply supports *Real life* and *Role playing* communities, and the manual should be read with this in mind:
Not all the features will make sense for all the applications.

.. note::

   The project is under heavy development and many things may break as the APIs are developed and rolled out!
