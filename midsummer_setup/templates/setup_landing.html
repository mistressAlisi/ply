Midsummer Setup

Welcome to Midsummer. This wizard will help you setup your instance of the Midsummer Community Manager. This setup takes approximately 10 minutes. Before you get started, make sure you have:

  • A PostgreSQL Server version 16 or newer*
  • A RabbitMQ Server version 3.13 or newer*
  • An Amazon S3 or S3 Compatible Service*
  • A Stripe account (If using the Registrar App)
  • A Mail Server (STARTTLS or SMTPS Supported)

* These services are included if using the Docker Compose File (PostgreSQL 16, RabbitMQ 3.13.7, MinIO RELEASE.2024-09-09T16-59-28Z)
